# 工具使用指南

本文件定义项目中各类工具的使用规范。

## Context7（文档检索）

专用于获取库/框架的最新官方文档与代码示例。

### 使用流程
1. 先调用 `resolve-library-id` 获取库 ID
2. 再调用 `get-library-docs`，需指定 `topic` 和 `tokens`
3. 在产出中标明库 ID、版本与引用出处

### 触发场景
- 新库上手
- 版本差异排查
- API 报错定位
- 生成调用样例

### 注意事项
- 不要用来做通用网页搜索
- 优先使用 Context7 获取库文档，而非 AI 内置知识

## Exa Search（实时网页检索）

专用于实时网页/新闻/博客/公告/社区问答等开放互联网搜索。

### 使用规范
- 控制关键词（3-6 个）
- 返回标题 + 简述 + URL + 抓取时间
- 说明排序依据（相关度/时效）
- 优先官方或高可信来源
- 可附原文链接

### 触发场景
- 官方站最新公告
- 漏洞/兼容性预警
- 发行说明
- 对比调研
- 训练截断后出现的新资料

### 注意事项
- 不要替代 Context7 做库文档检索
- 聚焦时效性与权威性

## 文档优先级

1. **用户明确要求** > 一切
2. **CLAUDE.md 与 rules/** > 其他规范
3. **README.md** > 外部最佳实践
4. **Context7、Exa 最新文档** > AI 内置知识
