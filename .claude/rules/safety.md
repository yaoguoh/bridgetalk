# 高风险操作确认

本文件定义需要用户明确确认的高风险操作类型。

## 需要确认的操作

执行以下操作前，必须获得用户明确确认：

### 文件操作
- 删除文件或目录
- 移动/重命名文件
- 批量文件改动

### 系统操作
- 系统配置变更
- 权限变更
- 环境变量变更

### 数据库操作
- 数据库结构变更（DDL）
- 批量数据修改/删除
- 数据迁移

### Git 操作
- `git commit`
- `git push`
- `git reset`
- `git rebase`
- 分支删除

### 网络操作
- 网络敏感请求
- 外部 API 调用（可能产生费用）

### 依赖操作
- 全局依赖安装
- 依赖版本升级
- 删除依赖

## 确认方式

1. 明确说明将要执行的操作
2. 列出影响范围
3. 等待用户确认后再执行

## 例外情况

以下情况可以直接执行：
- 读取操作（不修改数据）
- 本地开发环境的临时文件
- 用户明确授权的批量操作
